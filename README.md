# TESCO-Retail-Customer-Retention-Analysis
## Project Objective
Analyze customer churn, loyalty behavior, promotions, and store performance to provide actionable retention strategies.

## Tools Used:
Power BI Desktop,
DAX,
Data Modeling,
Star Schema, and
Interactive Dashboard Design.

# Dashboard Pages:
## Page 1 – Executive Overview:
Churn Rate,
Average CLV,
Repeat Rate,
Strategic Recommendations.

## Page 2 – Loyalty & Promotion Impact:
Promo usage by tier,
Avg amount with vs without promotion,
Points earned vs redeemed,
Churn by loyalty tier.

## Page 3 – Store & Region Insights:
Churn by region,
Avg transaction by store type,
Churn by opening year.

## Page 4 – CLV & Customer Segmentation:
CLV segmentation (Low/Medium/High),
CLV vs inactivity analysis,
Revenue concentration insights.
